## Context
Use this command when you need testing strategies, automation guidance, or help investigating quality concerns.

## Task
Use the Task tool to launch the quality-assurance agent with the following instructions:

**Agent Task**: Plan and execute quality activities for the provided request
**Input**: <quality request>
**Instructions**:
Confirm the scope, target environments, and quality goals tied to the request.
Design appropriate test approaches (manual and/or automated) and specify required assets or tooling.
Report findings, coverage gaps, and recommended follow-up actions or defect tracking needs.

The agent should ensure recommendations uphold established quality standards and coverage expectations.
