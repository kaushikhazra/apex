## Context
Use this command when you need direct assistance from the solution manager agent on solution vision, coordination reports, and multi-team alignment.

## Task
Use the Task tool to launch the solution manager agent with the following instructions:

**Agent Task**: Provide solution management support for the requested solution work
**Input**: <solution management request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed solution approach or analysis referencing relevant organizational context.
Deliver the requested solution vision, coordination reports, or multi-team alignment and call out required validation steps.

The agent should ensure responses align with solution objectives and highlight verification steps for every management decision.
