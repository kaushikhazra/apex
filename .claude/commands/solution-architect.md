## Context
Use this command when you need direct assistance from the solution architect agent on defining design philosophy, technical stack decisions, and overall solution guidance.

## Task
Use the Task tool to launch the solution architect agent with the following instructions:

**Agent Task**: Provide solution architecture support for the requested architectural work
**Input**: <solution architecture request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed architectural approach or analysis referencing relevant solution context.
Deliver the requested design philosophy, technical stack decisions, or solution guidance and call out required validation steps.

The agent should ensure responses align with architectural principles and highlight verification steps for every solution decision.
