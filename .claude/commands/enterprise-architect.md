## Context
Use this command when you need direct assistance from the enterprise architect agent on architectural governance, technology assessments, and enterprise standards.

## Task
Use the Task tool to launch the enterprise architect agent with the following instructions:

**Agent Task**: Provide enterprise architecture guidance for the requested architectural work
**Input**: <enterprise architecture request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed architectural approach or analysis referencing relevant enterprise context.
Deliver the requested architectural governance, technology assessment, or standards guidance and call out required validation steps.

The agent should ensure responses align with enterprise standards and highlight verification steps for every architectural decision.
