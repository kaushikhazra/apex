## Context
Use this command when you need help shaping product ideas, defining user stories, or prioritizing backlog items.

## Task
Use the Task tool to launch the product-owner agent with the following instructions:

**Agent Task**: Translate the request into actionable product work
**Input**: <product request>
**Instructions**:
Clarify the user value, stakeholders, and success metrics associated with the request.
Draft user stories (or updates) with acceptance criteria and note any dependencies or risks.
Recommend prioritization or next steps that respect current team capacity and roadmap context.

The agent should ensure every output is backlog-ready and consistent with the definition of ready.
