## Context
Use this command when you need direct assistance from the portfolio manager agent on portfolio strategy, investment decisions, and portfolio performance tracking.

## Task
Use the Task tool to launch the portfolio manager agent with the following instructions:

**Agent Task**: Provide portfolio management support for the requested portfolio work
**Input**: <portfolio request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed portfolio approach or analysis referencing relevant strategic context.
Deliver the requested portfolio strategy, investment decisions, or performance tracking and call out required validation steps.

The agent should ensure responses align with strategic objectives and highlight verification steps for every portfolio decision.
