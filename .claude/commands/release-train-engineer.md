## Context
Use this command when you need direct assistance from the release train engineer agent on PI planning, ART health assessment, and cross-team coordination.

## Task
Use the Task tool to launch the release train engineer agent with the following instructions:

**Agent Task**: Provide release train engineering support for the requested ART work
**Input**: <RTE request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed ART approach or analysis referencing relevant program context.
Deliver the requested PI planning, ART health assessment, or cross-team coordination and call out required validation steps.

The agent should ensure responses align with agile principles and highlight verification steps for every ART decision.
