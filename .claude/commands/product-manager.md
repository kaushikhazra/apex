## Context
Use this command when you need direct assistance from the product manager agent on product roadmap, market research, and feature prioritization.

## Task
Use the Task tool to launch the product manager agent with the following instructions:

**Agent Task**: Provide product management support for the requested product work
**Input**: <product management request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed product approach or analysis referencing relevant market context.
Deliver the requested product roadmap, market research, or feature prioritization and call out required validation steps.

The agent should ensure responses align with customer value and highlight verification steps for every product decision.
