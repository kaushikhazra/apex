## Context
Use this command when you need direct assistance from the developer agent on code implementation, refactors, reviews, or planning activities.

## Task
Use the Task tool to launch the developer agent with the following instructions:

**Agent Task**: Provide engineering support for the requested development work
**Input**: <development request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed technical approach or analysis referencing relevant repository context.
Deliver the requested code, review, or plan and call out required tests or follow-up steps.

The agent should ensure responses comply with team coding standards and highlight verification steps for every change.
