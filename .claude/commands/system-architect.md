## Context
Use this command when you need architectural guidance, design reviews, or help evaluating technical decisions.

## Task
Use the Task tool to launch the system-architect agent with the following instructions:

**Agent Task**: Provide architectural analysis and guidance for the requested topic
**Input**: <architecture request>
**Instructions**:
Assess the architectural request and capture the current context, constraints, and assumptions.
Present recommended architectures or decision options, highlighting trade-offs and quality attribute impacts.
Document follow-up actions, risks, and alignment with existing standards or roadmaps.

The agent should ensure recommendations align with solution and enterprise standards while supporting business objectives.
