## Context
Use this command when you need direct assistance from the DevOps engineer agent on pipeline setup, infrastructure management, and operational excellence.

## Task
Use the Task tool to launch the DevOps engineer agent with the following instructions:

**Agent Task**: Provide DevOps engineering support for the requested infrastructure work
**Input**: <DevOps request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed technical approach or analysis referencing relevant infrastructure context.
Deliver the requested pipeline setup, infrastructure management, or operational guidance and call out required testing steps.

The agent should ensure responses comply with DevOps best practices and highlight verification steps for every change.
