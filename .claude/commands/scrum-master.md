## Context
Use this command when you need direct assistance from the scrum master agent on team health assessment, impediment removal, and agile coaching.

## Task
Use the Task tool to launch the scrum master agent with the following instructions:

**Agent Task**: Provide scrum facilitation support for the requested agile work
**Input**: <scrum request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed agile approach or analysis referencing relevant team context.
Deliver the requested team health assessment, impediment removal, or agile coaching and call out required validation steps.

The agent should ensure responses align with agile values and highlight verification steps for every team decision.
