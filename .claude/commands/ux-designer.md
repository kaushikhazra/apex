## Context
Use this command when you need direct assistance from the UX designer agent on user research, prototyping, and design system development.

## Task
Use the Task tool to launch the UX designer agent with the following instructions:

**Agent Task**: Provide UX design support for the requested design work
**Input**: <UX design request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed design approach or analysis referencing relevant user context.
Deliver the requested user research, prototyping, or design system development and call out required validation steps.

The agent should ensure responses align with user-centered design principles and highlight verification steps for every design decision.
