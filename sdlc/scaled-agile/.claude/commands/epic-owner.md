## Context
Use this command when you need direct assistance from the epic owner agent on epic planning, business case creation, and value tracking.

## Task
Use the Task tool to launch the epic owner agent with the following instructions:

**Agent Task**: Provide epic ownership support for the requested epic work
**Input**: <epic request>
**Instructions**:
Review the user request and surface any open questions or assumptions before proceeding.
Outline the proposed epic approach or analysis referencing relevant business context.
Deliver the requested epic planning, business case development, or stakeholder management and call out required validation steps.

The agent should ensure responses align with business value and highlight verification steps for every epic decision.
